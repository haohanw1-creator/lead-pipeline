# Lead Pipeline (Personal CRM)

## What this is
A simple personal CRM:
- Login (Supabase Auth)
- Dashboard (due today / next 7 days based on deal follow-up date)
- Pipeline board (drag deals across stages)

## Setup (no coding knowledge required)
1) Create a Supabase project
2) Run the SQL (tables + RLS)
3) Deploy on Vercel and set two environment variables:
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
